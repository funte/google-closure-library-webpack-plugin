# exprements.angular
Convert Closure library to ES modules and use it with `Angular` and Webpack5 in `typescript`.  

To build and run this app with CLI:  
```sh
yarn install
# See help.
yarn run help
# Convert Closure library to ES modules.
yarn gen
yarn build
yarn start
```
